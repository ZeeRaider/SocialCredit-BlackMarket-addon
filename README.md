SocialCredit: BlackMarket

Illegal commerce, informants, inspections and state paranoia for SocialCredit servers

Turn your SocialCredit server into a living underground economy where players secretly trade goods, avoid inspections, report each other, and try not to attract the attention of the authorities.

This addon introduces a player-driven black market and enforcement system layered on top of the SocialCredit framework. It adds illegal commerce, state inspections, heat levels, informants, propaganda announcements and Discord broadcasts — turning your server into a chaotic political economy simulation.

What This Addon Adds

The core SocialCredit plugin introduces State Credits and a reputation-driven government system.

BlackMarket expands the world with illegal trade and surveillance mechanics.

Players can:

• Run hidden black market shops
• Buy and sell contraband
• Inform on suspicious players
• Avoid inspections and raids
• Launder suspicious activity
• Build underground trading networks
• Try not to get caught by the state

Meanwhile the server constantly broadcasts propaganda, warnings and economic directives to maintain the atmosphere.

The result is a dynamic social experiment where the economy, trust and paranoia evolve naturally.

Core Features
Black Market Shops

Players can create illegal shops using sign-based shop interfaces.

Features:

Locked shop inventories

State Credit payments

Player-owned commerce

Secure chest-based stock storage

Clean base layouts (sign driven shop interface)

The system prevents players from stealing stock directly from chests — purchases must go through the shop interface.

State Credit Integration

All transactions use SocialCredit's State Credit system.

Players must earn credits through gameplay and state contributions before they can participate in the underground economy.

Failure to meet economic expectations produces humorous system messages such as:

"Even under communism you must strive to meet the demands of the nation."

Heat / Suspicion System

Illegal activity generates Heat.

Players with high heat risk:

inspections

raids

confiscations

fines

public enforcement announcements

This makes black market trading risky and encourages stealthy behaviour.

Informant System

Players can secretly report suspicious activity.

Mechanics include:

citizen reporting

rewards for accurate reports

penalties for false accusations

anonymous intelligence reports

Informants help create player-driven policing and paranoia.

State Inspections & Raids

If illegal activity grows too large in an area the state may intervene.

Possible events:

inspections

raids

confiscation of goods

public enforcement broadcasts

These events create dynamic gameplay moments across the server.

Propaganda & Administrative Announcements

The plugin constantly broadcasts state messages and propaganda.

Examples:

civic notices

economic reminders

security directives

suspicious activity warnings

absurd bureaucratic statements

These are designed to make the server feel alive and slightly ridiculous.

Discord Integration

The addon integrates with the SocialCredit webhook system.

Events can be posted to Discord such as:

player arrivals

enforcement actions

state announcements

economic messages

This allows servers to extend the experiment outside Minecraft.

Gameplay Style

This plugin works best on servers that enjoy:

• emergent gameplay
• social experiments
• political satire
• player-driven economies
• chaotic roleplay

The goal is not strict enforcement — it's to create fun stories and unpredictable interactions.

Creating a Black Market Shop

Step 1
Place a chest and fill it with items you want to sell.

Step 2
Create a shop sign attached to the chest.

Example sign format:

[BlackMarket]
Diamond
Price: 15

Step 3
Players can now purchase the item using State Credits.

The chest remains locked to prevent theft.

Only the owner may add stock.

Security Rules

To maintain fair gameplay the plugin enforces:

• chest locking
• owner-only stock access
• sign-based purchases
• credit-verified transactions

Players cannot simply take items from shop chests.

Administration Tools

Admins can:

reload the plugin

trigger inspections

test Discord integration

open the Black Market control GUI

debug the system

Example command:

/blackmarketadmin reload

Installation

Install the SocialCredit core plugin

Download SocialCredit: BlackMarket

Place the jar into /plugins

Restart the server

The addon will automatically integrate with the SocialCredit API.
